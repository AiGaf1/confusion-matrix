ConfusionMatrix

src/index.js

Constructs a confusion matrix

new ConfusionMatrix(matrix: Array<Array<number>>, labels: Array<any>)
Parameters
matrix (Array<Array<number>>) The confusion matrix, a 2D Array. Rows represent the actual label and columns the predicted label.
labels (Array<any>) Labels of the confusion matrix, a 1D Array
Example
const CM = new ConfusionMatrix([[13, 2], [10, 5]], ['cat', 'dog'])

â–¸ fromLabels(actual, predicted, options = {})
src/index.js

Construct confusion matrix from the predicted and actual labels (classes). Be sure to provide the arguments in the correct order!

fromLabels(actual: Array<any>, predicted: Array<any>, options: object?): ConfusionMatrix
Parameters
actual (Array<any>) The predicted labels of the classification
predicted (Array<any>) The actual labels of the classification. Has to be of same length as predicted.
options (object? = {}) Additional options
Name Description
options.labels Array<any>? The list of labels that should be used. If not provided the distinct set of labels present in predicted and actual is used. Labels are compared using the strict equality operator '==='
Returns
ConfusionMatrix: Confusion matrix
